# Bridgeport 1.0.9

Bridgeport 1.0.9 adds automatic updates via Sparkle, and the project is now open source under GPL-3.0.

## Automatic updates

- Bridgeport now checks for updates using the Sparkle framework. A **Check for Updates…** item appears in the menu bar menu, and update checks run automatically on the standard Sparkle schedule.
- Updates are verified twice before installation: the DMG's EdDSA signature is checked against the public key embedded in the app, and the app's Developer ID signature and notarization are enforced by macOS. The update feed is served from the repository's `appcast.xml`.
- Releases 1.0.8 and earlier predate Sparkle, so this release must be installed manually; releases after 1.0.9 install through the updater.

## Open source

- The repository is now public under the GPL-3.0 license.
