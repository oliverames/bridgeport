# Bridgeport 1.0.12

Bridgeport 1.0.12 adds a user-facing option to keep the app icon visible in the Dock while running.

## Show or hide the Dock icon

- A new toggle in the Settings dashboard, "Show app icon in Dock," controls whether Bridgeport appears in the Dock or lives only in the menu bar.
- The setting defaults to off, preserving the existing menu-bar-only behavior.
- When enabled, the app uses a regular activation policy so the Dock icon stays visible while running.
- Opening the Settings window temporarily shows the Dock icon regardless of the setting, and closing it restores the user's preference.
- The preference persists in config.json and takes effect immediately when toggled, or on the next app launch.
