# Bridgeport 1.0.6

Bridgeport 1.0.6 prepares the project for a wider release with neutral first-run settings, an explicit security and privacy posture, gated continuous integration, and a verified macOS installation path.

## Release and installation

- Adds macOS 26 CI for Swift tests, isolated HTTP smoke tests, release builds, app-bundle assembly, and clean-install verification.
- Makes the release workflow call the reusable CI workflow before publishing.
- Adds full-history Gitleaks scanning with one fingerprint-specific exception for a confirmed documentation placeholder.
- Fixes DMG packaging so the mounted image contains `Bridgeport.app` and an Applications shortcut instead of exposing the app bundle's internal `Contents` directory.
- Adds a clean-install verifier for source-built app bundles and signed, notarized DMGs.

## Public-readiness defaults

- Uses `~/.config/bridgeport/connectors` as the neutral primary connector path for new installations.
- Leaves new Cloudflare domain and hostname fields blank and uses a generic profile label.
- Preserves explicit values already stored in existing user configuration.
- Replaces maintainer-specific paths, hostnames, and private deployment examples in the current documentation tree.

## Security and privacy

- Adds a security reporting policy, deployment hardening guidance, privacy disclosure, and detailed threat model.
- Documents the trust boundary around connector subprocesses, token storage, Cloudflare routing, OAuth, and local configuration.
- Adds a reproducible release checklist covering signing, notarization, Gatekeeper, clean installation, tests, and secret scanning.

## Verification

- All 44 Swift tests and all 12 isolated HTTP smoke checks pass on macOS.
- The release build and packaged app launch check pass.
- The signed DMG is accepted by Apple's notary service, stapled, accepted by Gatekeeper, and passes the isolated clean-install verifier.
- The release DMG SHA-256 is `12b3b534d5961860d36caca1221022656e9444b4bb9d730f810badcbd088b60b`.
- Full-history Gitleaks scanning reports zero unignored findings.

The repository remains private and all rights reserved until a source license is selected and the public-history review is resolved.
