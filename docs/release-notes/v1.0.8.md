# Bridgeport 1.0.8

Bridgeport 1.0.8 gives the app its bridge: new suspension-bridge artwork for the app icon and a matching hand-drawn menu bar icon.

## Icons

- The app icon now shows a single bold suspension bridge on the existing blue-teal gradient tile. The previous network-route overlay and node dots were removed so the bridge is the sole subject.
- The menu bar icon is a custom suspension-bridge template image drawn in code. SF Symbols has no bridge glyph (verified by probing `NSImage(systemSymbolName:)`), so this replaces the previous `app.connected.to.app.below` symbol and its `bolt` fallback. The icon renders at full strength while the daemon is running and dims when it is stopped, and adapts to light and dark menu bars automatically.

## Upgrade notes

- Finder and the Dock may cache the previous app icon until the new bundle is installed; running `killall Finder Dock` forces both to redraw icons.
